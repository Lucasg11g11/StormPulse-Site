⚙️ Requisitos do Painel

🪟 Para Windows

  • Windows 10 ou superior
  • .NET Framework 4.8 instalado
  • Permissão de Administrador
  • Antivírus e firewall configurado para permitir o StormPulse
  • NMAP instalado

🐧 Para Linux

  • Permissão de root (sudo)
  • Net-Tools instalado ➡️ "1-sudo apt update" "2-sudo apt install net-tools"
  • NMAP instalado
  • Rodar o exe com wine

🍎 Para MacOS

  • MacOS 11 Big Sur ou superior
  • Permissão de Administrador (sudo)
  • Xcode Command Line Tools instalado ➡️ "xcode-select --install"
  • NMAP instalado ➡️ "brew install nmap"
  • Permitir o StormPulse nas Preferências de Segurança
  • Rodar o exe com wine
